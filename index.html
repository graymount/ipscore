🔗 API 接口文档

使用以下curl命令访问我们的IP安全分析API

🏠 您的IP信息

获取当前访问者的IP地址信息，包括基本的IP数据。

curl https://ip-score.com/ip

📍 位置和ISP信息

获取您IP的详细地理位置信息和ISP数据，返回JSON格式。

curl https://ip-score.com/json

🎯 自定义IP位置查询

查询指定IP地址的地理位置信息和ISP数据。

curl https://ip-score.com/json -F 'ip="1.1.1.1"'

🛡️ 威胁情报检测

检查您的IP是否在各种黑名单和威胁情报库中，包括垃圾邮件列表等。

curl https://ip-score.com/spamjson

🎯 自定义IP威胁检测

检查指定IP地址的威胁情报信息。

curl https://ip-score.com/spamjson -F 'ip="1.1.1.1"'

📊 完整安全报告

获取您IP的完整安全评估报告，包括位置、ISP、威胁情报等所有信息。

curl https://ip-score.com/fulljson

🎯 自定义IP完整报告

获取指定IP地址的完整安全评估报告。

curl https://ip-score.com/fulljson -F 'ip="1.1.1.1"'
